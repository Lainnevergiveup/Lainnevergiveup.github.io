# 数据结构概论

数据结构是计算机存储、组织数据的方式。

## 逻辑结构

- **集合结构**：元素之间没有特定关系
- **线性结构**：元素之间一对一关系（如链表、栈、队列）
- **树形结构**：元素之间一对多关系（如二叉树、堆）
- **图结构**：元素之间多对多关系

## 存储结构

```c
// 顺序存储 - 数组
int arr[100];

// 链式存储 - 链表
struct Node {
    int data;
    struct Node *next;
};
```

## 基本操作

| 操作 | 说明 | 时间复杂度 |
|------|------|-----------|
| 查找 | 定位元素 | O(1) ~ O(n) |
| 插入 | 添加元素 | O(1) ~ O(n) |
| 删除 | 移除元素 | O(1) ~ O(n) |

> 选择合适的数据结构是算法设计的关键。
